# Báo cáo tiến độ tuần 01

## 1. Công việc đã thực hiện
- Tạo GitHub repository theo đúng cú pháp quy định.
- Khởi tạo README.md.
- Thêm file .gitignore.
- Tạo cấu trúc thư mục ban đầu cho đồ án.
- Xác định công nghệ sử dụng: PHP, MySQL, HTML, CSS, JavaScript, FullCalendar.


## 2. Công việc đang thực hiện
- Phân tích yêu cầu hệ thống.
- Thiết kế cơ sở dữ liệu.
- Xây dựng cấu trúc mã nguồn ban đầu.

## 3. Công việc dự kiến tiếp theo
- Tạo sơ đồ cơ sở dữ liệu.
- Tạo file schema.sql.
- Xây dựng giao diện và chức năng quản lý hoạt động.
- Tích hợp lịch làm việc bằng FullCalendar.

## 4. Khó khăn
- Cần giới hạn phạm vi chức năng để hoàn thành sản phẩm demo đúng thời gian.
